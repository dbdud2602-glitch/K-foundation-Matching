# 🌸 CUSHION MATCH - K-Beauty 쿠션 톤 매칭 서비스

올리브영 실시간 데이터 기반의 K-Beauty 쿠션/파운데이션 톤 매칭 웹사이트입니다.

## 📋 기능

### 톤 매칭 검색
- 사용 중인 쿠션의 브랜드와 톤 번호를 입력
- 다른 브랜드에서 동일하거나 유사한 톤을 찾기
- 매칭도를 퍼센트로 표시

### 고급 필터
- **커버력 필터**: Light, Medium, High
- **피니시 필터**: Natural, Semi-Matte, Matte, Glow, Dewy

### 포함된 브랜드 (15개)
- IOPE, LANEIGE, HERA, MISSHA, ETUDE
- VDL, AGE20S, ROUND LAB, ESPOIR
- JUNGSAEMMOOL, 3CE, TONYMOLY, WAKEMAKE, ABOUTONE

## 🎨 톤 시스템 가이드

### N (Neutral) - 뉴트럴톤
노란기와 붉은기 없는 자연스러운 톤
예: N21, N23, N25

### C (Cool) - 쿨톤
핑크빛이 도는 화사한 톤
예: C13, C17, C21, C23

### W (Warm) - 웜톤
노란색이 감도는 따뜻한 톤
예: W21, W23
